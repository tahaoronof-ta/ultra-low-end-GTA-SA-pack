# Installation Guide - Ultra Low-End SA Pack

## 🎮 Overview
This guide will help you install Ultra Low-End SA Pack for GTA San Andreas. The modpack is designed to improve FPS, reduce lag, and stabilize gameplay on low-end PCs.

---

## 📥 Step 1: Download

Download the latest version from the **GitHub Releases** page.

> Always use the latest stable release for best performance.

---

## 📦 Step 2: Extract Files

Use one of the following tools:
- WinRAR
- 7-Zip

Extract the downloaded `.zip` file to any folder.

---

## 📁 Step 3: Install Modpack

Copy all extracted files into your GTA San Andreas main directory:

Example path:C:\Program Files\Rockstar Games\GTA San Andreas\
Make sure to merge folders when prompted.

---

## ▶️ Step 4: Launch the Game

Run:gta_sa.exe
Enjoy improved performance and smoother gameplay.

---

## ⚙️ Recommended In-Game Settings

For best FPS performance:

- Frame Limiter: ON (recommended for stability)
- Resolution: Native monitor resolution or lower
- Draw Distance: Low / Medium
- Visual FX Quality: Low
- Anti-Aliasing: OFF
- Shadows: Low or OFF

---

## 🚀 Performance Tips

To maximize FPS:

- Close background applications (Chrome, Discord overlays, etc.)
- Disable Windows Game Bar
- Use Fullscreen mode (not Windowed)
- Update GPU drivers
- Avoid heavy graphics mods

---

## ⚠️ Compatibility

✔ GTA San Andreas v1.0  
✔ Single Player mode  

❌ Not tested with:
- SA-MP
- MTA
- Heavy ENB graphics packs

---

## 🧠 Notes

- This modpack does NOT change story or gameplay mechanics
- Designed only for performance optimization
- Some visual quality reduction is expected for better FPS
