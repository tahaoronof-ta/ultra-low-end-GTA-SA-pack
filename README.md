# 🚀 Ultra Low-End SA Pack

> A lightweight FPS optimization modpack for **Grand Theft Auto: San Andreas** designed for low-end PCs and smooth gameplay.

---

## 📌 Overview

**Ultra Low-End SA Pack** is a performance-focused optimization package that improves FPS, reduces stutters, and enhances stability in GTA San Andreas without changing the original gameplay experience.

This modpack is built for:
- Low-end PCs 💻
- Integrated graphics systems 🎮
- Old laptops and weak hardware ⚙️

---

## ✨ Key Features

- 🚀 Significant FPS boost (20%–80% on low-end systems)
- 🧊 Reduced lag and stuttering
- ⚡ Improved frame stability
- 🪟 Widescreen & modern resolution support
- 🔧 Fixed common crashes and bugs
- 🎮 Vanilla gameplay preserved
- 💡 Lightweight and fast setup
